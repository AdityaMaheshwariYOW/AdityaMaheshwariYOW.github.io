PCO Email AI Pipeline

Executive Summary

Built a scalable ML pipeline in Azure Databricks and PowerBI that processes ~10,000 daily emails to the PMO, extracting topics and sentiment to inform senior leadership in real time.

Technical Overview

  • Ingestion: Databricks → Azure Delta Lake; cleaned & anonymized
  • Models: RoBERTa sentiment, BERT-spam/toxicity, BERTopic clustering
  • Reporting: Automated daily HTML + PowerBI dashboards
  • Feedback Loop: Web app for analysts to correct labels & retrain models
