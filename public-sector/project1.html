Executive Summary

We built a proof-of-concept AI pipeline that ingests millions of emails addressed to the Prime Minister’s office, dynamically extracting key topics and sentiment to generate daily intelligence reports. This automated solution surfaces real-time insights into grassroots public concerns—helping senior leaders respond faster, improve transparency, and boost public trust.

Technical Overview

  • Data Ingestion & Storage: Emails are streamed into an Azure Delta Lake via Databricks, cleaned (anonymized, de-duped) and stored in parquet format.
  • Classification Models: ─ A suite of transformer classifiers (CardiffNLP RoBERTa for sentiment, BERT-based hate/toxicity, and custom spam detectors) flags each email as substantive, spam, or write-in.
  • Topic Modeling: BERTopic clusters substantive emails into emergent themes, extracting top keywords per cluster.
  • Reporting Pipeline: A scheduled Databricks job runs daily, writes aggregated metrics to PowerBI for dashboarding, and publishes static HTML reports to GitHub Pages.
  • Feedback Loop: Analysts use a lightweight web app to correct model outputs (language, spam, topic labels), with those corrections feeding back into periodic retraining jobs.
